Week 2

The make command, used to convert source code into machine code, is not a compiler by itself. It is a program that runs a compiler.

The compiler being used is clang.

If we use clang directly to compile a C program, it will by default output a file named a.out. To change the name of the output file, we use the -o option along with the desired name. Example:

clang -o hello hello.c

When using an external library, we need to reference that library to avoid errors. Example:

clang -o hello hello.c -lcs50

Compiling is only one of four steps involved in converting source code to machine code. The steps are:

  • Preprocessing
  • Compiling
  • Assembling
  • Linking

First, preprocessing occurs where header files in the code are copied and pasted into the program. Second, compiling converts the code into assembly code. Third, assembling converts the assembly code into machine code. Finally, linking combines the code from the included libraries with our code to produce a final executable file.

Arrays are a method of storing data consecutively in memory for easy access. Example:

int scores[3];
  
  scores[0] = 72;
  scores[1] = 73;
  scores[2] = 53;

To refer to any item in the array, such as the first score, we use:

scores[0]

We can also use a loop to initialize values for an array:

int scores[3];
  
  for (int i = 0; i < 3; i++)
  {
      scores[i] = get_int("Score: ");
  }

When mentioning an array in a function parameter, we can specify the type of variable stored along with array[]. Example:

float average(int length, int array[])
  {
      int sum = 0;
  
      for (int i = 0; i < length; i++)
      {
          sum += array[i];
      }
  
      return sum / (float)length;
  }

Strings are an example of an array; they store a sequence of char values together. The last item stored in a string array is \0, which indicates the end of the string.

We can count the length of a string by:

string name = get_string("Name: ");
  int n = 0;
  
  while (name[n] != '\0')
  {
      n++;
  }

Alternatively, we can use the C library #include which includes the strlen function:

int stringLength(string s)
  {
      int n = 0;
      while (s[n] != '\0')
      {
          n++;
      }
      return n;
  }

Using strlen from the library is more efficient than counting manually. Example of printing a string input:

#include 
  #include 
  #include 
  
  int main(void)
  {
      string s = get_string("Input: ");
      printf("Output: ");
      for (int i = 0; i < strlen(s); i++)
      {
          printf("%c", s[i]);
      }
      printf("\n");
  }

However, calling strlen multiple times within a loop can reduce efficiency. Instead, store the length in a separate variable:

int length = strlen(s);
  
  for (int i = 0; i < length; i++)
  {
      printf("%c", s[i]);
  }

We can also initialize another function within the loop:

for (int i = 0, n = strlen(s); i < n; i++)
  {
      printf("%c", s[i]);
  }

To convert a string to uppercase, we can use a loop:

#include 
  #include 
  #include 
  
  int main(void)
  {
      string s = get_string("Input: ");
      printf("Output: ");
      for (int i = 0, n = strlen(s); i < n; i++)
      {
          if (s[i] >= 'a' && s[i] <= 'z')
          {
              printf("%c", s[i] - 32);
          }
          else
          {
              printf("%c", s[i]);
          }
      }
      printf("\n");
  }

Alternatively, use the toupper function from the C library to convert lowercase characters to uppercase:

#include 
  #include 
  #include 
  #include 
  
  int main(void)
  {
      string s = get_string("Input: ");
      printf("Output: ");
      for (int i = 0; i < strlen(s); i++)
      {
          printf("%c", toupper(s[i]));
      }
      printf("\n");
  }

To make a program accept command-line arguments, use int argc, string argv[] instead of int main(void):

int main(int argc, string argv[])
  {
      // ...
  }

An example of a program that prints out user input from command-line arguments:

#include 
  #include 
  #include 
  
  int main(int argc, string argv[])
  {
      printf("%s\n", argv[1]);
  }

Here, argv[1] is the first string input by the user. argv[0] outputs the name of the file, while argv[2] would be the second string inputted.

argc represents the number of arguments passed at the CLI. For example, ./greet Vanuja has 2 arguments, so argc == 2.

To require exactly 2 arguments in the program, write:

int main(int argc, string argv[])
  {
      if (argc == 2)
      {
          printf("%s\n", argv[1]);
      }
      else
      {
          printf("Error: Enter 2 arguments\n");
      }
  }
