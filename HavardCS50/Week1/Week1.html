Week 1

A compiler is a program that converts one language into another, most often used in the context of source to machine code.

The CLI in VS Code is the terminal where commands can be typed to perform certain functions in VS Code without the use of a mouse or GUI.

Some examples of commands used in the terminal include:

  • code hello.c (opens or creates a new file named hello.c)
  • make hello (acts as a compiler that converts the source code within hello.c into machine code)
  • ./hello (runs the machine code file called hello)
  • clear (clears all the code present in the terminal)
  • cd (changes the current directory/folder)
  • ls (lists the files within a directory)
  • cp (copies files within a directory)
  • mkdir (creates a new directory)
  • mv (moves or renames files or directories)
  • rm (removes a file)
  • rmdir (removes a directory)

Example of C code:

#include <stdio.h>

int main(void)
{
    printf("hello, world\n");
}

#include <stdio.h> is a command that tells the compiler that the library <stdio.h> should be used, which is a header file allowing us to use the printf function along with other things.

printf is an example of a function; functions take in arguments and return values. In this case, it displays 'hello, world' on the terminal.

\n creates a new line after hello, world.

Another example of a function in C is get_string, which is defined in the <cs50.h> library:

get_string("What's your name? ");

This prompts the user "What's your name?". To save the user's answer within a variable, we need to assign the function to a variable and define the type of variable from right to left:

string answer = get_string("What's your name? ");

Other examples of types of variables include:

  • int (used for storing integers)
  • char (for storing a single character/letter)

Adding const before declaring a variable means that the value within the variable cannot be reassigned.

Examples of functions used along with their types of data include:

  • get_int
  • get_char
  • get_float
  • get_double
  • get_long

Once we have stored data within a variable, we can display/use that data within a function using %s as a placeholder:

printf("Hello, %s\n", answer);

Examples of corresponding placeholders used to include a type of variable within a function include:

  • %i (for integers)
  • %c (for characters)
  • %f (for floating-point numbers)
  • %d (for integers)
  • %li (for long integers)

Conditionals are expressions that evaluate to either true or false. In C, an example of a conditional is an if statement:

if (x > y)
{
    printf("x is greater than y\n");
}
else if (x < y)
{
    printf("y is greater than x\n");
}
else
{
    printf("x and y are equal\n");
}

Within a function, we can use operators to compare variables/data:

  • || (represents OR)
  • && (represents AND)

Loops are sequences of instructions that are repeated until a given condition is met. The different types of loops in C include:

  • while loop (repeats the instructions while the condition is true)
  • for loop (repeats the instructions for a certain number of iterations)
  • do while loop

Example of a while loop:

int i = 0;

while (i < 3)
{
    printf("Meow\n");
    i++;
}

Example of a for loop:

for (int i = 0; i < 3; i++)
{
    printf("Meow\n");
}

There are 3 sections within the for loop:

  • The definition of a variable
  • The condition set for the variable
  • Incrementing the variable

We can create a function by first defining its name, the parameters it takes, and writing the instructions within the function. Example:

void meow(void)
{
    printf("Meow\n");
}

The function must be defined before it is called, or we can copy the function's prototype (in this case, void meow(void)) and paste it above int main(void).

If we want the function to take in a value, we can define it within the function instead of using void. Example:

void meow(int n)
{
    for (int i = 0; i < n; i++)
    {
        printf("Meow\n");
    }
}

To use the function, we type:

meow(3);

This causes the program to print Meow 3 times.

If the function returns an integer, the function must be defined with int. Example:

int add(int a, int b)
{
    return a + b;
}

void represents a function with no return value, while it can be replaced by the return data type if the function does have a return value.

A do while loop is another example of a loop in C:

int n;

do
{
    n = get_int("Enter number: ");
}
while (n < 1);

Note: The code above is incorrect. We cannot refer to the variable n as a string inside the curly brackets due to scope. Hence, we also have to declare it above the do while loop:

int n;

do
{
    n = get_int("Enter number: ");
}
while (n < 1);

A comment in C can be made by typing //.
