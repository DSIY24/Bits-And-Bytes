Blink

Within the Arduino are pins in which we can attach various input or output devices / circuits. However out of all the pins pin 13 is unique as it is already connoted to an in-built LED. In this project we will learn how to turn on and off this LED and make it blink in fixed intervals.

First within the void setup() we have to type pinMode( 13, OUTPUT). pinMode takes in two parameters, first is which pin is being declared ( in this case it is pin 13 ). The second is whether we want to use this pin as an INPUT or OUTPUT. Commands are case sensitive.

Once we have declared the pin we then have to turn on the LED, the function we have to use is digitalWrite and write digitalWrite(13, HIGH). 13 tells which pin to send the signal to, HIGH tells to set the voltage high.

Then when we click the upload arrow on the top right it sends the code to the Arduino which should then turn the LED on.

We can then type LOW into the second parameter in digitalWrite to turn off the LED.

To make the LED blink we have to add a a delay function. delay only take one perameter with it being time in milli seconds.
